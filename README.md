# DtMoney

## Descrição do Projeto

Aplicação desenvolvida no modulo 2 na trilha de ReactJS do ignite.

#### Pré-requisitos:

- Git
- `yarn` ou `npm`

### Rodando Projeto: </br>



  Clone o repositório:  
  `https://github.com/RodrigoAndrade085/dt-Money.git`
  
  Acesse a pasta do projeto:  
  `cd dt-money`
  
  Instale as dependências:  
  `yarn ou npm install`

  Rode o projeto  
  `yarn start`
